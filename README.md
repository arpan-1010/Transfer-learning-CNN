# 🧠 Transfer Learning CNN (VGG16 - PyTorch)

This repository uses **Transfer Learning using VGG16** to classify Fashion MNIST images using PyTorch.

---

## 🚀 Features

- Uses pre-trained **VGG16 model**
- Applies transfer learning (freezes feature layers)
- Custom dataset handling with PyTorch
- Image preprocessing & transformations
- Training and evaluation loops
- GPU support (if available)

---

## 📂 Dataset Used : 
- Fashion MNIST
